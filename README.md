# JavaCar – Sistema de Gestión de Alquiler de Vehículos

## Descripción del proyecto
JavaCar es una aplicación en Java para gestionar el alquiler de vehículos como coches, motos y furgonetas. Permite calcular el precio del alquiler y los ingresos totales según los días.

El proyecto está hecho para practicar Programación Orientada a Objetos (POO) con clases, herencia e interfaces.

---

## Contexto
Este proyecto forma parte de una práctica académica donde trabajas como programador junior dentro de un equipo con una arquitectura ya definida. Tu tarea es completar funcionalidades y hacer que los tests automáticos pasen correctamente.

También incluye un enfoque de sostenibilidad mediante la etiqueta ambiental de los vehículos (DGT).

---

## Tecnologías usadas
- Java
- Programación Orientada a Objetos (POO)
- Git / GitHub
- JUnit (tests)
- GitHub Actions (CI/CD)

---

## Funcionalidades principales
- Gestión de vehículos (coches, motos y furgonetas)
- Cálculo del precio de alquiler por días
- Cálculo de ingresos totales
- Filtrado de vehículos por precio máximo
- Asignación de etiqueta ambiental según características del vehículo
